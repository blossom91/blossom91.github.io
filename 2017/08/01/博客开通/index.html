hexo框架说明

hexo依赖于node与git

新建目录

_config.yml

参数 描述
title 标题
subtitle 副标题
description 网站描述
author 名字
language 语言
timezone 时区
url 网址
root 根目录
permalink 文章的 永久链接 格式
theme 当前主题名称
deploy 扩展
default_layout 预设布局(默认post)

部署说明

Front-matter

参数 描述
layout 布局
title 标题
date 日期
updated 更新日期
comments 开启评论功能
categories 分类(有时序性)
tags 标签(无顺序性)
toc theme特有 目录显示
mathjax theme特有 数学符号
layout: timeline theme特有 时间线
